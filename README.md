# 📊 Task 3: Customer Funnel Analysis

## 📌 Project Overview
This project focuses on analyzing customer behavior using a funnel approach. The goal is to understand how customers are retained or lost (churn) in a telecom dataset.

---

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib

---

## 📂 Dataset
- Telco Customer Churn Dataset

---

## 📊 Analysis Performed
- Created a customer funnel:
  - Total Customers
  - Active Customers
  - Churned Customers
- Visualized data using bar charts
- Calculated retention rate and churn rate

---

## 📈 Insights
- Most customers are retained by the company
- A noticeable number of customers have churned
- Reducing churn can significantly improve business performance

---

## 🚀 Conclusion
This project demonstrates how data analysis can help businesses identify customer drop-off and improve retention strategies.
